# Falke Handel - Next.js Website

A modern Next.js website based on the Figma design for Falke Handel, featuring global export solutions.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js app directory with pages and layout
- `components/` - React components (Header, Hero, CookieBanner)
- `app/globals.css` - Global styles and CSS variables

## Features

- Responsive design matching the Figma prototype
- Cookie consent banner with localStorage persistence
- Modern, clean UI with custom color scheme
- TypeScript support
- Next.js 14 with App Router

## Build

To build for production:

```bash
npm run build
npm start
```
